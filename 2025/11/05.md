- 3．フォントで検索して『adobe Fonts』を選択

- 4．今回は『平成丸ゴシック』を選択

- 5．フォントファミリーを作成して追加

- 6.Webプロジェクトに追加　からコードをコピー

- 7. 作成している.htmlの<header></header>の中に貼り付け

- 8.詳細からフォントファミリーをコピー

　今回はfont-family: heisei-maru-gothic-std, sans-serif;

- 9.css 内で利用したいタグに設定

  　今回は

  　h1, h2, p {

      font-family: heisei-maru-gothic-std, sans-serif;

  　}
  
 -  line-height: 23px;　行間を設定
