> ## わかったこと
- git stash
  - 作業を一時避難する
  - →ステージとワークツリーで作業していたファイルを一時的にstashに避難させて
  - ステージとワークツリーを変更前に戻して別の作業を行えるようにする
- 最新の作業を復元する
  - git stash apply
- ステージの状況も復元する
  - git stage apply --index
